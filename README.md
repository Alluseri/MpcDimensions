# MpcDimensions
A handy dimension auto-picker for Mapartcraft.

## How to installl
1. Install TamperMonkey (surely you already have it): [Chrome](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo), [Firefox](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)
2. Open [this link](https://github.com/Alluseri/MpcDimensions/raw/refs/heads/main/MpcDimensions.user.js)
3. Click "Install" (or "Reinstall")
4. Be happy! (hopefully)
